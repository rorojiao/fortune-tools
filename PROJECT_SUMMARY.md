## 🔧 技术实现

### 前端架构
```
fortune-tools/
├── index.html          # 主页（5个工具导航）
├── bazi.html          # 八字算命
├── tarot.html         # 塔罗牌占卜
├── name.html          # 姓名测分
├── dream.html         # 周公解梦
├── daily.html         # 今日运势
├── css/
│   └── common.css     # 公共样式
├── js/
│   ├── common.js      # 公共函数（打赏、分享、Toast）
│   ├── bazi.js        # 八字算法
│   └── tarot.js       # 塔罗牌数据
└── README.md
```

### 核心功能模块

**common.js (公共组件):**
- `getTipBoxHTML()` - 打赏组件（钱包地址展示）
- `getShareBarHTML()` - 分享按钮（微信/微博/Twitter）
- `copyWallet()` - 复制钱包地址
- `showToast()` - 消息提示
- `seededRandom()` - 可重现的伪随机数生成器

**bazi.js (八字核心):**
- 天干地支数组
- 五行映射表
- 纳音表
- 十神计算
- 年/月/日/时柱推算函数
- 五行统计与分析

**tarot.js (塔罗数据):**
- 22张大阿尔卡纳完整数据
- 56张小阿尔卡纳（4花色×14牌级）
- 正逆位解读文案
- 抽牌算法
